 Changes Made

  1. CSS Utility Classes (src/styles/index.css)

  - Typography utilities: text-display-xl, text-display-lg, text-display-md, text-display-sm, text-body-lg, text-body-md, text-body-sm, text-caption
  - Content width utilities: content-narrow (768px), content-medium (1024px), content-wide (1152px), content-full (1280px)
  - Banner/Image utilities: banner-container, banner-image, banner-overlay, banner-overlay-subtle, image-showcase, image-showcase-sm, showcase-image, showcase-image-lg

  2. Tailwind Config (tailwind.config.js)

  - Added semantic color tokens: semantic.success, semantic.warning, semantic.error, semantic.info
  - Added text tokens: text.primary, text.secondary, text.tertiary, text.muted
  - Added surface tokens: surface.base, surface.elevated, surface.overlay

  3. Theme Constants (src/styles/theme.js)

  - Added iconSizes: xs (14), sm (16), md (20), lg (24), xl (32)
  - Added gridGaps: tight (gap-4), default (gap-6), spacious (gap-8)

  4. New Components

  - BannerImage.jsx: Reusable banner component with loading states, error handling, and optional overlay
  - Skeleton.jsx: Loading skeleton component with presets for Text, Card, Avatar, and Image

  5. Page Updates

  All pages updated with:
  - Removed inline styles: Replaced with Tailwind classes
  - Fixed text contrast: Changed text-dark-400/500 to text-dark-200/300 for better readability
  - Standardized grid gaps: Changed varied gaps (4, 5, 6, 8, 12) to consistent gap-6 (default) or gap-8 (spacious)
  - Standardized content widths: Using content-medium and content-full classes
  - Standardized icon sizes: Using iconSizes.sm/md/lg/xl from theme
  - Used BannerImage component: HomePage, TEDPage, TrainingPage now use the new component
